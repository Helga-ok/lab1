# lab1
Тема. Знакомство с HTML  и разработка простого веб-приложения.
Задание (вариант 10). Реализовать возможность перемещения и поворота произвольной HTML страницы (через iframe) средствами CSS 2D transforms (при помощи управляющих кнопок и при помощи мыши). 
Вращение реализовано на кнопках «←» «→», перемещение реализовано для мыши.
